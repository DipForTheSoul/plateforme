# Guide d'utilisation — Administration ForTheSoul

> Pour Didier. Tout se fait depuis **forthesoul.ch/admin** (téléphone ou ordinateur).

## Se connecter

1. Ouvrir forthesoul.ch → **Connexion** (en haut à droite).
2. E-mail : `welcome@forthesoul.ch` + votre mot de passe.
3. Vous arrivez sur le **tableau de bord** : les compteurs encadrés en orange
   signalent ce qui attend votre validation.

## Valider ou refuser une expérience (le geste quotidien)

**Admin → Soumissions.** Chaque carte montre le titre, le praticien, la date, le
lieu et la description.

- **✓ Valider & publier** : l'expérience est en ligne immédiatement, le
  praticien reçoit un e-mail (avec votre message si vous en avez écrit un).
- **✕ Refuser** : l'expérience reste hors ligne, le praticien reçoit un e-mail
  avec votre message — écrivez toujours un mot pour expliquer.
- Le champ « Message au praticien » est optionnel mais recommandé.
- Une expérience **récurrente** : votre décision s'applique à toutes ses dates.

## Mettre une expérience en avant (Sélection du moment)

**Admin → Soumissions → Historique** : « ☆ Mettre en top listing » place
l'expérience dans la « Sélection du moment » de la page d'accueil.

## Valider un·e praticien·ne

**Admin → Praticien·nes.** Nouvelles fiches en haut. Validez après votre
rencontre/échange — la fiche devient publique et la personne peut publier.

## Ajouter un lieu

**Admin → Lieux → Ajouter un lieu.** Entrez l'adresse complète (rue, NPA,
ville) : la position GPS est trouvée automatiquement. Si « Adresse
introuvable », précisez l'adresse.

## Crédits (publications)

- Les praticiens achètent leurs packs seuls (carte bancaire).
- Pour un **virement reçu** : **Admin → Crédits** → choisir le praticien, le
  nombre de crédits, une note (« Virement reçu le … ») → Attribuer.
- L'historique complet est affiché dessous (achats, attributions, consommations).

## Newsletter

**Admin → Newsletter** :
- **Importer** : collez vos contacts (une ligne par personne :
  `email, prénom, nom, tag1|tag2`).
- **Tags** : modifiez les centres d'intérêt directement dans la liste.
- **Exporter** : bouton « Tout exporter » ou un tag précis → fichier CSV prêt
  pour MailerLite.

## Statistiques

Le tableau de bord affiche les pages les plus vues sur 30 jours. La mesure se
fait **sans cookies** : aucune bannière n'est nécessaire sur le site.
